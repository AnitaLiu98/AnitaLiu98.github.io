# 题目描述
请实现一个函数，把字符串 s 中的每个空格替换成"%20"。
来源：Leetcode
题目详情：https://leetcode-cn.com/problems/ti-huan-kong-ge-lcof/

# 算法实现
## 1.一行流（咳咳咳
这么做的话这个题目对于python来说真的意义不是很大......
### Python3 实现
```
class Solution:
    def replaceSpace(self, s: str) -> str:
        return s.replace(' ','%20')
```

## 2. 一次循环，逐步扩展-时间复杂度 O(N^2)
### Python3 实现
```
class Solution:
    def replaceSpace(self, s: str) -> str:
        index=0
        while index<len(s):
            if s[index]==' ':
                s = s[0:index]+'%20'+s[index+1:]
                index+=3
            else:
                index+=1
        return s

```
