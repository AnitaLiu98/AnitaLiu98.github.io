# 归并排序
## 算法流程
- 将数组A[1,n]排序问题分解为A[1,n/2]和A[n/2+1,n]排序问题
- 递归解决子问题得到两个有序的子数组
- 将两个有序子数组合并为一个有序数组
## 分而治之的一般步骤
分解原问题；解决子问题；合并问题解
## 伪代码
MergeSort(A,left,right)

```

输入：数组A[1..n],数组下标left，right
输出：递增数组A[left..right]
if left>=right then
    return A[left..right]
end
mid<-[(left+right)/2]
MergeSort(A,left,mid)
MergeSort(A,mid+1,right)
Merge(A,left,mid,right)
return A[left..right]

```
